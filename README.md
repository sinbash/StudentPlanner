# StudentPlanner

StudentPlanner is a web application designed to help students manage their academic activities, including schedules, tasks, and personal planning.

## Purpose
The project aims to provide a simple and efficient platform for students to organize their study routines and access essential features like authentication, profile management, and academic tracking.

## Features
- User authentication (login, signup, password reset)
- Profile management
- Dashboard for student activities
- Organized frontend using HTML, CSS, and JavaScript
- Backend powered by Flask

## Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- Git & GitHub for version control

## Status
Currently under development (Frontend completed, Backend in progress).
